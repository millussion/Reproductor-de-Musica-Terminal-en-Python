# Reproductor-de-Musica-Terminal-en-Python
Reproductor de Musica Terminal en Python, se necesita importar e instalar pygame. instálalo con pip install pygame en terminal.
Las canciones van en la carpeta canciones, para agregar más musica se debe hacer otro elif cargando la canción.



Si al descargar no sale la carpeta .venv hay que crearla con el comando en la terminal python -m venv .venv

Para ejecutar el codigo python main.py
